# 📊 Netflix Data Analysis

This project explores the Netflix dataset using Python to uncover trends, clean data, and visualize insights.

---

## 🚀 Project Overview

The goal of this project is to perform **Exploratory Data Analysis (EDA)** on Netflix content and understand patterns such as:
- Growth of content over the years
- Distribution of movie durations
- Content types (Movies vs TV Shows)
- Top countries producing Netflix content

---

## 🧹 Data Cleaning

- Handled missing values in columns like `director`, `cast`, `country`, and `rating`
- Converted `date_added` to datetime format
- Extracted year from date for time-based analysis
- Cleaned and converted `duration` into numeric format
- Split and exploded country data for accurate analysis

---

## 📊 Visualizations

The project includes:

- 📌 Bar Chart → Movies vs TV Shows  
- 📌 Pie Chart → Content Ratings Distribution  
- 📌 Line Plot → Number of Releases Over Time  
- 📌 Histogram → Distribution of Movie Durations  
- 📌 Horizontal Bar Chart → Top 10 Countries  

---

## 📈 Key Insights

- Most Netflix movies are between **80–120 minutes**
- Content production increased significantly after **2015**
- Movies dominate the platform compared to TV Shows
- United States and India are among the top content producers
